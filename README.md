# BrainBoost

Your free AI-powered learning companion for students from Class 1 to Class 12.

## Features

- **AI-Generated Worksheets**: Create personalized worksheets tailored to your class and subject
- **24/7 AI Chat Support**: Get instant, detailed explanations for any topic
- **Smart Explanations**: Understand complex concepts with easy-to-follow AI explanations
- **Practice Questions**: Test your knowledge with AI-generated practice questions
- **All Subjects Covered**: Math, Science, English, Hindi, Computer, Physics, Chemistry, Biology & more
- **100% Free**: No hidden costs or subscriptions

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
